This is a place to put my sex-related stuff so that it doesn't intrude on people who'd rather not see it, and in order that filtering software can treat it separately. As such, it will inevitably be something of a hodgepodge.

Sex, sexuality, and all that goes along with them are great interests of mine. I think it's a pity that so many cultures treat sex as abhorrent or not susceptible to discussion: like anything else, the more we learn about it, the better we can approach it and work with it.

The information varies a bit in quality, but this site is a great starting point for research.
